Mapeamento e caracterização

A Pesquisa NUI compreendeu o desenvolvimento e aplicação de uma metodologia para mapeamento e caracterização dos Núcleos Urbanos Informais, realizada por meio de uma combinação de estratégias: consultas a agentes municipais, verificações remotas com utilização de imagens de satélite, utilização de dados secundários, e em alguns casos, visitas de campo (vistorias).

Atenção

Essa base de dados não possui validade como cadastro municipal. Consulte o levantamento oficial de núcleos urbanos informais com as secretarias municipais.

Superfícies de probabilidade

Com o intuito de auxiliar na identificação de Núcleos Urbanos Informais, foi desenvolvida uma metodologia que indica a probabilidade de presença de NUI nos territórios analisados. Essas superfícies de probabilidade podem ser utilizadas como um plano adicional de informação para subsidiar trabalhos de campo e/ou para análise da qualidade de bases de dados existentes. Os modelos foram construídos utilizando a técnica ‘regressão logística’ em dados secundários de fontes distintas (Censo Demográfico 2010, Cadastro Único, OpenStreetMap e outras) integrados em uma grade celular com 100m de lado (compatível com a Grade Estatística do IBGE).
